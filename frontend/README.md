# Yeoman UI Frontend

>**IMPORTANT**: Use Node v10 (works with 10.24.1) and 'npm` (not Yarn !) to build frontend part.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
